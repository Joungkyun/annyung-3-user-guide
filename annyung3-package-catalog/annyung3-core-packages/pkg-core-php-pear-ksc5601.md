# php-pear-KSC5601

## Description:

KSC5601 pear 패키지는 한글과 관련된 변환 및 체크에 대한 method를 제공한다.

## Features:

1. 한글 여부 체크 \(KSC5601-87, KSC5601-92/KSX1001, UTF-8\)
2. KSX1001, UTF-8 변환
3. UHC, NCR 변환
4. UHC, UCS2 변환
5. 한글 substr 처리
