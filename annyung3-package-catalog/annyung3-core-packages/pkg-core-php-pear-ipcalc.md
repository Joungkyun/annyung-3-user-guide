# php-pear-ipcalc

## Description:

IP 계산과 서브네팅을 지원한다.

## Features:

이 API는 libipcalc에서 제공하는 IP 연산 및 서브네팅 API를 pure php code로 구현한 것이다. [php-krisp](pkg-core-php-krisp.md)의 IP 연산 및 서브네팅 function들과 호환 된다.
