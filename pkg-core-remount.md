# remount

### Description:

Mount/Unmount remote file system

### Features:
1. autofs와 비슷한 기능이나, 좀더 직관적으로 remote fs를 다루고 싶을 경우 사용
2. systemd를 사용하지 않고, 기존의 init script를 사용한다.

### Reference:
* 구성 파일
 * /etc/rc.d/init.d/remount
 * /etc/sysconfig/remount

### Dependencies:
* None

### Sub Packages:
* None

### Releated Packages:
