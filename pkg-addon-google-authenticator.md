# google-authenticator

### Description:
login 시에 2단계 인증을 가능하게 해주는 PAM 인증 모듈.

Google OPT와 연동이 가능하며, iOS와 Android App으로 지원합니다.


### Features:
* HMAC-Based One-time Password(HOTP) 지원
* Time-based One-time Password(TOTP) 지원
